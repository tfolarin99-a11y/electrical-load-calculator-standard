How to Use This Tool

A complete step-by-step guide for electrical engineers, technicians and contractors.
Work through the tabs from left to right for a full professional calculation workflow.

Taewoo Digital Techniques LLC — Standard Edition
1

📋 Project Tab

Start every project here. Fill in the project name, client, location, engineer name, drawing number and applicable standard (IEC, BS 7671, NEC etc.). Set your supply voltage, system type (3-phase or 1-phase), power factor, and installation country. These details appear on your final report.

💡 Tip: Selecting the correct voltage here updates ALL calculations in every module automatically.
2

📝 Loads Tab

Enter every electrical load in the building. Choose from 8 built-in presets (Office, House, School, Hotel, Warehouse, Hospital, Retail, Factory) to get started quickly. For each load enter: description, category, quantity, unit wattage, power factor, demand factor and phase assignment.

💡 Tip: Demand Factor = how often a load runs at full power. Lighting = 1.0, Sockets = 0.4, HVAC = 0.8–0.9, Motors = 0.6–0.8.
3

📊 Demand Tab

Select your building type card to apply the correct demand factor, or type a custom value. The tool calculates Connected Load, Demand Load, kW, kVA, kVAR and a Design kVA with your growth margin applied. The phase load bar shows how load is distributed across R, Y and B phases.

💡 Tip: Always size main equipment using Design kVA — it includes your growth margin.
4

⚡ Current Tab

Click "Import kW from Demand" to pull your demand figure in automatically. The tool applies the correct formula for your system type and calculates line current, apparent kVA, reactive kVAR and a Design Current with 125% safety factor already applied.

💡 Tip: Always use Design Current (×1.25) for cable and breaker sizing — never the raw line current.
5

〰 Cables Tab

Click "Import Design Current" to auto-fill. Select cable material, insulation and installation method (9 options from A1 to F). Set ambient temperature and grouped circuits. The tool applies full IEC derating and recommends the correct cable size with a voltage drop check inline.

💡 Tip: Higher ambient temperature and more grouped circuits both reduce cable capacity — the tool handles all derating automatically.
6

🔌 Breakers Tab

Click "Import Line Current" to fill in the full load current. Select load type — motor and continuous loads trigger the 125% rule. Choose breaking capacity based on your switchgear location. The tool recommends the correct MCB, MCCB or ACB.

💡 Tip: For motor loads, always add a DOL or Star-Delta starter in your panel — the breaker alone is not sufficient motor protection.
7

📉 Volt Drop Tab

Click "Import from Cables Module" to fill current, length and material automatically. Select cable size and your applicable standard (IEC 3%, IEC 5%, NEC 2.5%, BS 4%). The tool shows voltage drop in volts and percent with a visual PASS/FAIL gauge.

💡 Tip: If voltage drop fails, try a larger cable size or split the circuit into shorter runs.
8

🔋 Generator Tab

Click "Import from Demand" to pull in your demand kW. Set your motor starting margin (default 25%) and spare margin (default 20%). Select fuel type, supply rating (Standby/Prime/Continuous) and enclosure type. The tool recommends the correct standard genset size.

💡 Tip: Use Standby (STP) rating for emergency backup. Use Prime (PRP) if the generator will run as the main power source.
9

🔆 Inverter Tab

Click "Import from Demand" or enter your backup load manually. Set backup duration, battery voltage, efficiencies and depth of discharge. The tool calculates the required inverter size in kVA and total number of batteries needed.

💡 Tip: LiFePO4 batteries support 90%+ depth of discharge, significantly reducing the number of batteries needed vs lead-acid.
10

🔄 Transformer Tab

Click "Import from Demand" to fill load kVA. Set loading factor (default 80%), primary/secondary voltages, vector group, cooling type and impedance. The tool recommends the nearest standard transformer size and calculates the fault level at the secondary terminals.

💡 Tip: The fault level result sets the minimum breaking capacity (Icu) your downstream switchgear must have.
11

📄 Report Tab

Click "Generate Report" to produce a full professional summary — project details, load schedule, demand results, and all equipment recommendations. Click "Print / Save PDF" and choose "Save as PDF" in your browser's print dialog.

💡 Tip: Enable "Background graphics" in the print dialog for the best-looking PDF with all colours preserved.

✅ Recommended Workflow — Standard Edition

  • Always start at the Project tab — set system voltage and type correctly before anything else
  • Use load presets as a starting point, then customise rows to match your actual project
  • Work left to right through the tabs — each module feeds into the next
  • Use the "Import" buttons in each tab to carry results across modules automatically
  • Always size cables and breakers using the 125% Design Current, not the raw line current
  • Always check voltage drop — especially for cable runs over 30m
  • Size the transformer at 80% loading maximum to allow for future load growth
  • The fault level from the Transformer module sets the minimum Icu rating for your breakers
  • Save your PDF report before closing the browser — data is not saved between sessions
  • For official project submissions, have all calculations reviewed and stamped by a registered engineer
